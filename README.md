# NSE Option Chain Tracker

A Next.js + React app that fetches NSE Option Chain data every 30 seconds and displays it in a table.

## Features

* Auto refresh every 30 sec
* Fetch NSE option chain data
* Simple table UI

## Setup

```bash id="h3k2lm"
npm install
npm run dev
```

Open http://localhost:3000

## Notes

* NSE API may require headers or proxy
